## SCTL
基於OpenCC轉換星際公民翻譯文本，簡體中文為繁體中文(含慣用語替換)的工具  
專案連結: [GitHub](https://github.com/teinterence/SCTL)  
下載連結: [Google雲端硬碟](https://drive.google.com/drive/folders/1QucHfHuk0-ZhFGisN11-ESRv7ZA3USSs?usp=sharing)  
中文文本來源: [星際公民漢化組](https://citizenwiki.cn/Localization)  
中文聊天編碼表來源: [SC漢化盒子](https://github.com/StarCitizenToolBox/app)
